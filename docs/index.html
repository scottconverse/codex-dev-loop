Codex Dev Loop

A durable development workflow system for Codex desktop: memory vaults, goal cards, approval queues, automations, and review dashboards.

Get Started Read Manual

What It Does

Durable Memory

Keep project brief, preferences, decisions, and open loops in a reviewable `.codex-loop/` vault.

Strong Goals

Turn vague requests into goal cards with scope, constraints, verification, and status.

Approval Queue

External actions wait for explicit approval: pushes, merges, deploys, messages, and production changes.

Review Surfaces

Generate dashboards and artifacts so user and Codex can inspect the same object.

Install

git clone https://github.com/scottconverse/codex-dev-loop.git
cd codex-dev-loop
powershell -ExecutionPolicy Bypass -File .\installers\install.ps1

Use In Codex

Use $codex-dev-loop for this repo. Initialize the dev loop vault.
