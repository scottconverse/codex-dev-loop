This thread is for recurring Codex desktop workflows.

Candidate patterns:

- watch a PR every 30 minutes until CI passes
- summarize review comments each morning
- check deployment status until a preview is ready
- review `.codex-loop/open-loops.md` weekly
- draft follow-ups from approved sources without sending

When sharing a pattern, include:

- target
- cadence
- stop condition
- approval boundary
- expected output
