# Approval Gates

Codex must ask before:

- sending messages, emails, or comments externally
- publishing, deploying, merging, or releasing
- deleting user data or running destructive filesystem operations
- changing production configuration or secrets
- spending money or making purchases
- taking irreversible action in authenticated external systems

Codex may usually proceed with:

- reading project files
- local edits requested by the user
- local tests, builds, and previews
- drafting messages or PR descriptions
- creating local artifacts for review
